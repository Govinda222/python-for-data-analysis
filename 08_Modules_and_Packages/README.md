# 📦 Modules and Packages in Python

## 📌 Objective

Learn how Python organizes reusable code using modules and packages.

---

## 📚 Topics Covered

- What is a Module?
- Creating Custom Modules
- Importing Modules
- Different Import Methods
- Python Standard Library
- Packages
- __init__.py
- Organizing Python Projects

---

## 📂 Files

| File | Description |
|------|-------------|
| modules_and_imports.ipynb | Introduction to modules and importing custom modules |
| standard_library.ipynb | Exploring Python's built-in standard library |
| package_demo.ipynb | Creating and using Python packages |
| maths.py | Custom module containing basic arithmetic functions |
| __init__.py | Package initialization file |

## 🎯 Key Takeaways

After completing this section I learned:

- How to create reusable Python modules.
- Different ways to import functions and modules.
- How Python's Standard Library saves development time.
- The purpose of packages.
- Why `__init__.py` exists.
- How larger Python projects are organized into multiple files.

---

## 💡 Real-World Applications

Modules and packages are used in almost every Python project.

Examples include:

- `pandas`
- `numpy`
- `matplotlib`
- `scikit-learn`

In future projects, I will organize my own code into modules instead of writing everything inside a single Python file.
